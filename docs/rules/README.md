# 规则库 — OpenStratCore

本目录是**游戏规则的仓内真源**。OpenStratCore 的内置规则集是项目自有的一套实时六角格陆战规则，
以**数据形式**承载（rules-as-data）——不依赖任何外部文档。这些是**默认/内置资源**；图形化规则编辑器
可在其基础上增改并导出新的规则资源。

## 真值优先级（出现分歧时）
1. **`config/tables/*.json`** — 机器可读的裁决表，**引擎直接消费**，是数值的权威真源。每个文件带
   `source`（内部章节/附录标号）与 `verified` 标志。
2. **`config/rules.default.json`** — 规则总表（速度、时长、射程、各 provider 配置）。
3. **`combat-model.md`** — 四条裁决流水线（直瞄对车/对人、间瞄、防空、雷场）的综合说明。
4. **`coverage.md`** — 规则落地的**进度真源**（哪些机制/表已实现、已核验）。

> 数值核验采用**双人独立交叉核验**：两名作者各自录入同一张表，逐格比对一致后标 `verified:true`；
> 不一致或未复核的标 `verified:false`，落地前需再次交叉核验。

## 给实现者的关键全局事实
- **1 六角格 ≈ 200 米**（由附5 速度换算自洽）。
- **排级算子**：1 个算子 = 1 个排，含 1-4 车 / 班。"班/车数"既是血量也是火力倍率输入。
- **两套骰**：对车辆用 **2d6**（随机数 2-12）；对人员/间瞄/防空的"结果修正"用 **1d6**。裁决表行号即随机数。
- **钻石型裁决**：间瞄(附2)、防空(附3.2) 的结果绕 **随机数=7** 对称——7 最不易命中，2/12 最致命。
- **75 秒**是绝大多数状态转换的基本节拍；**150s** 压制/间瞄飞行；**300s** 间瞄爆炸/冷却；**25s** 同格交战间隔。
